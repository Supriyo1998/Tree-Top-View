# Tree-Top-View
Tree: Top View in Java using Queue and Tree Map

Java code to print the top view of the tree. (Using Queue and Tree Map)

Example :
    ##      1
    ##       \
    ##        2
    ##         \
    ##          5
    ##         / \
    ##        3   6
    ##         \
    ##          4
              
           
          Top View : 1 -> 2 -> 5 -> 6
       i.e, From the top only nodes 1,2,5,6 will be visible.
